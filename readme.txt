== Description ==

This plugin can help you convert data from bbpress to DW Question & Answer

No data will be lost and delete.

<img src="https://plugins.svn.wordpress.org/my-bbpress-to-dw-qa-converter/assets/logo.png">
== Installation ==

1.Upload my-bbpress-dwqa-converter to the /wp-content/plugins/ directory
2.Activate the plugin through the 'Plugins' menu in WordPress
3.Log In to your WordPress Dashboard and go to Tools >> BBpress to DW Q&A then click to convert to convert all data from bbpres to DW Question & Answer.
